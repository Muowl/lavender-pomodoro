# Lavender Pomodoro

A gentle Pomodoro desktop app built with Tauri, React, TypeScript, and Bun.

## Theme

This project uses the lavender palette below as the base for the interface:

- `#f5e6e8`
- `#d5c6e0`
- `#aaa1c8`
- `#967aa1`
- `#192a51`

The current UI direction is soft, rounded, and calm:

- serif-forward display typography
- light and dark theme support
- minimal single-panel layout
- English UI copy

## Starter Features

- focus, short break, and long break modes
- working countdown timer
- start, pause, reset, and advance controls
- full stats reset with `Fresh Start`
- cycle tracking for long breaks

## Run

```bash
cd lavender-pomodoro
bun install
bun run dev
```

## Build

```bash
cd lavender-pomodoro
bun run build
```
